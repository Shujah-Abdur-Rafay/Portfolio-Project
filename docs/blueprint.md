# **App Name**: Sylvia Ogweng Digital Presence

## Core Features:

- Website Replication: Replicate the existing website based on the provided HTML code using Next.js components.
- SEO Meta Tags: Add meta tags from the original HTML source (title, description, og:title, og:description, og:image, twitter:card) to improve SEO.
- Navigation Menu: Implement a navigation menu that allows users to easily access different sections of the website.
- Contact Form Integration: Set up a contact form using a service like Formspree or Netlify Forms to allow users to send messages.
- Image Optimization: Optimize all images on the website to ensure fast loading times.

## Style Guidelines:

- Background color: Dark gray (#272727) to maintain a modern and sophisticated feel, with other grays used throughout, especially #2a2a2a for secondary backgrounds.
- Primary color: Light grayish yellow (#DCD7D2) for text to ensure readability against the dark background.
- Accent color: Bright, neon-ish yellow (#F0FF8C) to be used on hover for buttons and links, to create eye-catching interactivity
- Headline font: 'Sohne Semibold' (sans-serif) for a clean and modern look, specifically for headlines and titles.
- Body font: 'Times New Roman' (serif) for main content and paragraphs, providing a classic and readable style.
- Maintain a grid layout with a content width of 1100px to ensure consistency and readability across different screen sizes.
- Use elegant and font-awesome icons for visual representation of different website functionalities.
